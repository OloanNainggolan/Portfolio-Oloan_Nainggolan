# Assets Guide - Portfolio Website

## Folder Structure

### Main Images (assets/)
- **profile.jpg** - Foto profil untuk floating card (ukuran: 120x150px)
- **hero.jpg** - Gambar hero section (ukuran: 600x500px, landscape)
- **about.jpg** - Gambar about section (ukuran: 500x500px)
- **CV-Samuel.pdf** - File CV untuk download
- **Portfolio.pdf** - File portfolio lengkap untuk download
- **favicon.ico** - Icon browser tab (16x16px)

### Project Images (assets/projects/)
- **project1.jpg** - Screenshot Cinema Booking System (ukuran: 400x240px)
- **project2.jpg** - Screenshot E-Commerce Platform (ukuran: 400x240px)
- **project3.jpg** - Screenshot Inventory Management (ukuran: 400x240px)
- **project4.jpg** - Screenshot School Management (ukuran: 400x240px)
- **project5.jpg** - Screenshot HR Management (ukuran: 400x240px)

## Cara Menambahkan Gambar

### Opsi 1: Upload Gambar Sendiri
1. Siapkan foto/gambar dengan ukuran yang disarankan
2. Copy file ke folder yang sesuai
3. Pastikan nama file sama dengan yang ada di HTML

### Opsi 2: Gunakan Placeholder
Gunakan layanan seperti:
- https://placeholder.com
- https://picsum.photos
- https://unsplash.com

Contoh URL untuk placeholder:
```
Hero Image: https://picsum.photos/600/500
Profile: https://picsum.photos/120/150
Projects: https://picsum.photos/400/240
```

## Tips Gambar
- Format: JPG untuk foto, PNG untuk logo/icon
- Ukuran file: Maksimal 500KB per gambar
- Kompres gambar sebelum upload untuk performa lebih baik
- Gunakan aspect ratio yang sesuai

## Recommended Tools
- **TinyPNG** - Kompres gambar
- **Canva** - Edit dan resize gambar
- **Remove.bg** - Hapus background foto profil
